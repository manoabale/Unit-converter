# Unit-converter
 📏 Unit Converter

A simple and fast **Unit Converter** built with **HTML**, **CSS**, and **JavaScript**.  
This tool allows users to convert between common length units such as meters, kilometers, centimeters, miles, and feet.

Perfect for beginners learning DOM manipulation, math operations, and interactive UI design.

--l

## 🚀 Features

### 🔢 Supported Units
- **Meters (m)**
- **Kilometers (km)**
- **Centimeters (cm)**
- **Miles (mi)**
- **Feet (ft)**

### ⚡ Core Features
- Instant conversion  
- Clean and responsive UI  
- Beginner‑friendly JavaScript  
- Works on CodePen, GitHub Pages, Netlify, and any static hosting

## 💻 Technologies Used

- **HTML5** – Structure

- **JavaScript (Vanilla)** – Conversion logic
